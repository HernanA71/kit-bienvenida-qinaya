Soporte Nivel 1

1. Problemas de Inicio y Pantalla

La pantalla se queda en negro después de arrancar

Causa: Incompatibilidad de controladores gráficos (NVIDIA/Legacy).

Solución paso a paso:

  1. En el menú de GRUB (al arrancar), presiona la tecla 'e' para editar.
  2. Busca la línea que comienza con linux y termina con quiet splash.
  3. Agrega al final de esa línea un espacio y: nomodeset
  4. Presiona F10 para iniciar el sistema.
  5. Una vez dentro, instala los controladores privativos desde el menú "Controladores adicionales".

No aparece el menú de QinayaLinux y entra directo a Windows

Causa: Prioridad del gestor de arranque de Windows.

Solución:

  1. Inicia Windows y abre la Terminal (PowerShell/CMD) como Administrador.
  2. Escribe y ejecuta el siguiente comando:
bcdedit /set {bootmgr} path \EFI\qinayalinux\grubx64.efi

2. Configuración de BIOS y Boot Menu

Teclas de acceso rápido por marca

Presiona repetidamente al encender el equipo:

  • HP: F9 o Esc
  • Dell, Lenovo, Acer, Toshiba: F12
  • Asus: Esc o F8
  • Configuración BIOS: F2, F10 o Del

Error "Secure Boot" o "Invalid Signature"

Solución: Entra a la BIOS (F2/F10), busca la pestaña Security o Boot y cambia Secure Boot a Disabled o Desactivado.

3. Red y Conectividad

No detecta redes WiFi

  1. Verifica el interruptor físico de WiFi o la combinación de teclas (Ej: Fn + F2).
  2. Conecta cable Ethernet para descargar controladores específicos.
  3. En el menú de QinayaLinux, busca "Controladores adicionales" y activa el driver de red (Broadcom/Realtek).

4. Estación de Trabajo (STB)

¿Cómo restaurar el STB Qinaya?

Si el sistema está bloqueado o no arranca:

  1. Con un alfiler, presiona el botón dentro del orificio AV (Pinhole).
  2. Conecta la energía manteniendo presionado el botón por 5 segundos.
  3. En el menú Recovery, selecciona "Apply update from EXT" y elige el archivo update.zip.
